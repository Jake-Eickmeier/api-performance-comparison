# python-fastapi

A very basic API made using Python and FastAPI. To run locally, follow the below steps:

- Create a python virtual environment

`python -m venv .venv`

- Install required dependencies

`pip install -r requirements.txt`

- Run the application (instructions for developer mode, port 8000)

`fastapi dev app/main.py`

- OpenAPI documentation will be available at http://localhost:8000/docs
