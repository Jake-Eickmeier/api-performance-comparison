# java_spring

A very basic API made using Java and Spring. To run locally, follow the below steps:

- Build the application (Optionally, add `-DskipTests` to skip the tests)

`mvn clean install`

- Run the application (port will be 8001 per application.properties)

`mvn spring-boot:run`

- OpenAPI documentation will be available at http://localhost:8001/swagger-ui/index.html
